# Menu Maker by Qwenta – Projet OpenClassrooms

## Description

Ce projet fait partie de ma formation d’intégrateur web chez OpenClassrooms.
Il porte sur la planification et la préparation du développement d’un site web, sans intervention directe sur le code.

La mission consiste à analyser le projet à partir des éléments fournis (maquette, spécifications fonctionnelles, User Stories), à organiser le travail de développement et à définir une solution technique cohérente et justifiée.

## Compétences visées

- Analyse fonctionnelle et compréhension du besoin client
- Rédaction de spécifications techniques
- Découpage du projet en tâches (Kanban)
- Réalisation d’une veille technologique ciblée
- Structuration et documentation d’un projet web

## Contenu du repository

- Spécifications techniques
- Tableau Kanban
- Veille technologique
- Présentation du cadrage

⚠️ Aucun développement n’est présent dans ce repository. Le projet est volontairement limité à la phase de conception.
